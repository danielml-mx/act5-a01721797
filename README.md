# Act 5 - Canvas Simulator

## Lo que el app hace

Esta app es un simulador de Canvas primitivo hecho en Swift.

## ¿Qué API usa?
Para el API, [cree mi propio documento JSON](https://danielml.mx/clases.json) en mi sitio web con los datos de las materias que cursamos en 5to semestre de ITC.

## ¿Cómo lo puedo correr?

### MacOS:
Dentro de `Sources/` encontrarás el directorio con los archivos de trabajo así como el directorio con la información del proyecto de XCode.

### Linux
Desde el directorio base, corre `xtool dev`. [Descubre aquí más sobre xtool](https://xtool.sh).
